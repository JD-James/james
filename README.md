# James Flat Design System Demo

A Next.js App Router project with custom flat-design style guide implemented.

## Included pages
- `/` homepage with hero, cards, stats, pricing
- `/product` product catalog
- `/about` about with principles
- `/signin` auth shell

## Components
- `Navbar` (shared header)
- `Button` variant system (primary / secondary / outline)
- `Card` flat feature card
- `ProductCard` pricing product item
- `Input` accessible input design

## Tech stack
- Next.js App Router
- Tailwind CSS
- TypeScript
- Lucide icons

## Setup

1. Run `npm install`
2. Run `npm run dev`

> Note: This environment may not have `npm`/`npx` available; install dependencies in your local machine.
